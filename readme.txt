Delete the Custom Fields from your postmeta table where assigned posts no longer exist.

== Description ==

Delete all Custom Field entries assigned to posts that do not exist in the posts table.

Navigate to Settings -> Remove Unused CFs and delete all postmeta for non-existing posts.

== Installation ==

1. Upload the DX Remove Unused Custom Fields plugin to your blog and activate it.
2. Go to Settings -> Remove Unused CFs
3. Click 'Delete' to drop the unrelated fields.

== FAQ ==

= Is it permanently removing the data? =

Yes, the data would be removed from the database so use at your own risk.
